jquery.lazyany

Com o jquery.lazyany é possível atrasar o carregamento de qualquer elemento da página, permitindo que se carregue apenas o que o usuário de fato está vendo.

Exemplo

$('.selector').lazyany(function(el) {
  // do anything whith $(el)
});

API

selector.lazyany(<Function>)

Executa a função passada como parâmetro quando o(s) elemento(s) do selector aparecerem na tela.

A função recebe como parâmetro o elemento que está aparecendo na tela.

$('img').lazyany(function(img) {
  img.style.display = 'block';
});

selector.lazyany('destroy')

Remove o registro de lazyload nos elementos, dessa forma o callback não será mais disparado quando o elemento entrar no viewport do usuário.

selector.lazyany(<Object>)

É possível passar alguns parâmetros de configuração para o lazyany.

<Int> offset

Distância, em pixels, antes do elemento ser exibido, mas que o callback é executado.

$('img').lazyany({ offset: 300 }, function(img) { ... });

<Function> callback

A função callback pode ser passada pelo objeto de configuração ou como parâmetro.

$('img').lazyany({ offset: 300, callback: function(img) { ... } });

O parâmetro callback será ignorado se ele for enviado junto com o plugin

<String> plugin

Plugin é uma forma de facilitar e centralizar tarefas simples e recorrentes, como por exemplo, lazyload de imagens.

$('img').lazyany({ offset: 300, plugin: 'image' });

Para registrar um plugin:

$.fn.lazyany.plugins.pluginName = function(el) { ... }

O plugin image já vem registrado. Para funcionar, a tag img deve possuir o atributo data-original:

<img data-original="http://upload.wikimedia.org/wikipedia/commons/8/81/Sky_over_Washington_Monument.JPG" class="lazyany" width="620" height="465" title="Original uploader was Andypham3000 at en.wikipedia">
